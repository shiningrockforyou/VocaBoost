# VocaBoost — 조교(TA) 지원 가이드

학생들에게 가장 자주 발생하는 문제를 **선생님 계정**에서 처리하는 간단한 안내입니다.
좋은 소식: 최근 업데이트 이후 대부분의 문제는 자동으로 해결되거나 여러분이 직접 처리할 수 있습니다 —
맨 아래에 적힌 몇 가지 경우만 저(David)에게 알려주시면 됩니다.

---

## 먼저 알아둘 4가지

1. **"채점 실패(Grading Failed)"는 이제 대부분 자동으로 복구됩니다.** 네트워크가 잠깐 끊겨 채점 오류가
   떠도, 앱이 채점 결과를 자동으로 되살립니다. 학생에게 잠시 기다리거나 버튼을 다시 누르거나 **페이지를
   새로고침**하라고 하세요 — 보통 해결됩니다. **새로고침 후에도 계속 실패할 때만** 저에게 알려주세요.

2. **이제 이의제기(Challenge)를 직접 승인할 수 있습니다** — 더 이상 저를 기다릴 필요 없습니다. (1번 참고)

3. **학생의 점수를 바꾼 뒤에는 반드시 학생이 새로고침해야 합니다.** 이의제기를 승인하거나 설정을 바꾸면
   학생 화면이 **저절로 갱신되지 않습니다.** 바뀐 점수나 열린 다음 Day를 보려면 **페이지를 새로고침**하라고
   안내하세요.

4. **반(클래스)에서 단어장(리스트)을 추가하거나 제거하지 마세요.** 모든 반은 이미 올바른 단어장
   (Ascent, Summit 등)으로 세팅되어 있습니다 — 학생은 대시보드의 선택기(selector)에서 원하는 단어장을 고르기만
   하면 됩니다. 단어장 배정을 바꾸면 학생의 진도가 어긋날 수 있으니, 반의 단어장이 이상해 보이면 직접 바꾸지
   말고 **저에게 알려주세요.**

---

## 자주 발생하는 문제 → 해결 방법

### 1. "맞는 답인데 오답 처리됐어요" (채점 이의제기)
**가장 흔한 문제이며, 이제 여러분이 직접 해결할 수 있습니다.**
1. 학생에게 결과 화면에서 해당 답 옆의 **Challenge(이의제기)** 버튼을 눌러 제출하라고 하세요.
2. **Gradebook(성적부)** → 해당 학생의 시험 열기 → 대기 중인 이의제기를 찾으세요.
3. 동의하면 **Accept(승인)**, 아니면 **Reject(거절)** 를 누르세요.
4. 승인하면 그 답이 정답 처리되고 점수가 다시 계산되며 — 그 결과 통과 점수에 도달하면 — 다음 Day로
   **자동으로 넘어갑니다.**
5. 학생에게 **새로고침**해서 반영된 내용을 확인하라고 하세요.

*참고: 거절하면 학생은 30일 동안 이의제기 토큰 1개를 잃습니다. 정말 틀린 경우에만 거절하세요.*

### 2. "채점 실패(Grading Failed)" / 시험이 채점되지 않아요
- **페이지를 새로고침하고 다시 제출**하라고 하세요. 앱이 네트워크 끊김을 자동 복구하므로 보통 이렇게 해결됩니다.
- *"페이지를 새로고침하고 다시 제출하세요"* 같은 안내가 뜨면, 그게 **맞는 조치**입니다 — 새로고침하라고 하세요.
- **새로고침 후에도 계속 실패할 때만 저에게 알려주세요.**

### 3. "진도가 Day 1로 초기화됐어요" / "단어장이 사라졌어요"
- 진도가 사라진 게 아닙니다 — 올바른 단어장이 화면에 표시되기만 하면 됩니다.
- 먼저 학생에게 대시보드의 **단어장 선택기**에서 올바른 단어장을 고르라고 하세요.
- 그래도 진도가 돌아오지 않으면 → **저에게 알려주세요.** (단어장을 직접 추가/제거하지 마세요 — 위 4번 참고.
  반은 이미 올바르게 세팅되어 있습니다.)

### 4. "Day가 안 넘어가요" / "시험 후 막혔어요"
- 먼저 확인: 정말 **통과**했나요? (반마다 통과 점수가 있습니다, 예: 92%.) 통과 점수보다 살짝 낮으면 그냥
  **다시 시험(retake)** 을 봐야 합니다 — 정상이며 오류가 아닙니다.
- 이의제기 답 때문에 통과 점수에 못 미친 거라면 → **1번(이의제기 승인)** 으로 처리하세요.
- **통과**했는데 안 넘어가면 → 먼저 **새로고침**하라고 하세요.
- 새로고침해도 통과한 시험에서 계속 막혀 있으면 → **저에게 알려주세요.**

### 5. "로딩 화면에서 멈춰요"
- **다른 기기나 브라우저**로 시도하고 새로고침해 보라고 하세요.
- 특정 Day에서만 계속 멈추면 → **저에게 알려주세요** (어느 Day인지, 가능하면 스크린샷도 함께).

### 6. 학생이 엉뚱한 단어장에 있어요 / 자꾸 엉뚱한 단어장으로 들어가요
- 올바른 단어장은 모든 반에 이미 들어 있습니다 — 학생이 대시보드의 **단어장 선택기**에서 원하는 단어장을
  고르기만 하면 됩니다. 그곳에서 올바른 단어장을 선택하게 하세요.
- 특정 학생의 **기본(default) 단어장**을 바꿔야 하면 → **저에게 알려주세요.**

## SUMMIT으로 전환하기 (ASCENT을 끝낸 후)

ASCENT은 **1,600단어**입니다. 1,600단어가 모두 학습되면 ASCENT을 끝낸 것입니다:
- **ADV 반 (하루 80단어): Day 20 이후** → **Day 21**에 SUMMIT으로 전환.
- **FINAL 반 (하루 100단어): Day 16 이후** → **Day 17**에 SUMMIT으로 전환.

(SUMMIT은 800단어 → 하루 100단어면 약 **8일**, 80단어면 약 **10일**.)

**전환 방법:** SUMMIT은 **이미 모든 반에 추가되어 있습니다** — 따로 배정할 필요가 없습니다. 학생이 준비되면
대시보드의 단어장 선택기에서 **SUMMIT**을 고르라고 하세요. SUMMIT은 자체 Day 1부터 새로 시작합니다 —
새 단어장이므로 정상입니다.

**전환 시점 신호:** ASCENT을 끝낸 학생은 **새 단어가 더 이상 나오지 않습니다**(복습만 나옴). 그것이 선택기에서
SUMMIT을 고르게 할 시점이라는 신호입니다.

## ADV에서 FINAL로 옮긴 학생 (같은 ASCENT 단어장)

이 학생들은 두 가지가 달라집니다 — 미리 안내해 주세요:

1. **하루 학습량이 80 → 100단어로 늘어납니다** (FINAL의 속도). 같은 단어장, 더 많은 하루 분량.
2. **진도가 이어지지 않습니다** — FINAL 반으로 옮긴 학생은 ADV에서 여러 Day를 끝냈더라도 **ASCENT을
   Day 1부터 다시 시작**합니다. 기존 ADV 진도는 그대로 남아 있습니다.
   **이건 버그이며, 알려주시면 바로 고쳐드립니다** — 학생 이메일과 있어야 할 Day만 보내주시면 진도를
   옮겨드리겠습니다.

## 저에게 알려주셔야 할 경우
학생의 **이메일 + 반 + 어느 Day**, 그리고 가능하면 **스크린샷**과 함께 알려주세요:
- **ADV에서 FINAL로 옮겨**(또는 반이 바뀌어) 진도가 Day 1로 초기화된 학생 — 있어야 할 Day를 알려주시면 바로
  진도를 옮겨드립니다.
- **새로고침 후에도** 계속되는 "채점 실패".
- 새로고침해도 풀리지 않는, **통과한** 시험/Day에서의 막힘.
- 특정 Day의 **로딩 화면 멈춤**.
- 특정 학생의 **기본 단어장** 변경.
- **잘못되거나 누락된 데이터**로 보이는 모든 것 (점수가 안 맞음, 중복 기록 등).

---

*대부분의 문의는 1번(채점 이의제기)과 2번(채점 끊김)이며, 둘 다 이제 여러분이 직접 해결할 수 있습니다.
애매할 때는 일단 학생에게 새로고침을 시켜보세요 — 의외로 많은 문제가 그것으로 해결됩니다.*
