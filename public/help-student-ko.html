1시작하기

계정 만들기

  1. vocaBoost 웹사이트에 접속합니다.
  2. Sign Up 버튼을 클릭합니다.
  3. 이름(Full Name), 이메일, 비밀번호를 입력합니다.
  4. 역할(Role)에서 Student를 선택합니다.
  5. 졸업 예정 연도(Graduation Year)와 월(Month)을 선택합니다.
  6. Create Account 버튼을 클릭하면 가입이 완료됩니다.
간편 가입

Continue with Google 버튼을 클릭하면 Google 계정으로 빠르게 가입할 수 있습니다.

로그인하기

  1. vocaBoost 웹사이트에 접속합니다.
  2. Log In 버튼을 클릭합니다.
  3. 가입할 때 사용한 이메일과 비밀번호를 입력합니다.
  4. Continue 버튼을 클릭하면 대시보드로 이동합니다.

클래스 참가하기

  1. 선생님에게 참가 코드(6자리)를 받습니다.
  2. 대시보드에서 참가 코드 입력란을 찾습니다.
  3. 참가 코드를 입력합니다.
  4. Join Class 버튼을 클릭하면 클래스에 참가됩니다.
참가 코드 형식

참가 코드는 ABC123 같은 6자리 영숫자입니다. 대소문자를 정확히 입력하세요.

2대시보드 이해하기

로그인하면 가장 먼저 보이는 화면이 대시보드입니다. 여기서 모든 학습을 시작할 수 있습니다.

대시보드 구성 요소

항목 설명
My Classes 참가한 모든 클래스 목록
Assigned Lists 각 클래스에 할당된 단어장들
Progress 각 단어장의 학습 진행률
Streak 연속으로 학습한 일수

진도 표시 (Mastery Squares)

단어장 옆에 보이는 7개의 작은 사각형이 진도 표시입니다.

  • 초록색: 학습을 완료한 날
  • 회색: 아직 학습하지 않은 날

7칸이 모두 초록색이 되면 일주일 연속 학습 완료입니다.

연속 학습의 효과

매일 조금씩 꾸준히 학습하면 기억에 훨씬 오래 남습니다. 연속 학습 기록(Streak)을 유지해보세요.

결과 확인

학습 시작하기

  1. 대시보드에서 학습할 단어장을 찾습니다.
  2. Start Session 버튼을 클릭합니다.
  3. 오늘의 새 단어를 플래시카드로 학습합니다.
  4. 새 단어 테스트를 봅니다.
  5. 이전에 배운 단어들을 복습합니다. (Day 1은 복습 없음)
  6. 복습 테스트를 봅니다.
  7. 결과를 확인하고 세션을 완료합니다.
세션 중 나가도 괜찮습니다

학습 중에 나가도 진행 상황이 저장됩니다. 다시 돌아오면 이어서 할 수 있습니다. 단, 테스트 중에 나가면 테스트는 처음부터 다시 봐야 합니다.

자동 학습량 조절

vocaBoost는 여러분의 성적에 따라 자동으로 학습량을 조절합니다:

  • 성적이 좋으면: 새 단어를 더 많이 배웁니다
  • 성적이 낮으면: 새 단어를 줄이고 복습을 늘립니다

시스템이 여러분에게 맞는 속도로 조절해줍니다.

4플래시카드 사용법

플래시카드는 새 단어를 배우는 가장 효과적인 방법입니다.

카드 구성

앞면 뒷면
영단어 + 품사 영문 정의 + 한국어 뜻 + 예시 문장

조작 방법

  • 카드 뒤집기: 카드를 클릭하거나 Space 또는 Enter 키
  • 발음 듣기: 스피커 아이콘을 클릭

학습 진행

플래시카드 학습 중 다음 버튼들을 사용할 수 있습니다:

  • I know this: 이미 아는 단어일 때 (큐에서 제거)
  • Not Sure: 아직 익숙하지 않을 때 (큐에 유지)
  • Take Test: 학습을 완료하고 테스트로 진행
효과적인 학습 방법

1. 먼저 영단어를 보고 뜻을 떠올려보세요
2. 카드를 뒤집어 정답을 확인하세요
3. 예시 문장을 읽으며 사용법을 익히세요
4. 발음을 들으며 소리내어 따라해보세요

PDF 다운로드

학습한 단어들을 PDF로 다운로드해서 인쇄할 수 있습니다. 플래시카드 학습 화면의 메뉴에서 Download PDF를 선택하세요.

5테스트 유형

Multiple Choice 테스트

선택지 중 정답을 고르는 방식입니다. 선택지 개수는 선생님 설정에 따라 4~10개입니다.

  1. 영문 정의를 읽습니다.
  2. 선택지 중 맞는 단어를 클릭합니다.
  3. 모든 문제를 풀면 Submit Test를 클릭합니다.
문제 이동

화면 하단의 번호를 클릭하면 특정 문제로 바로 이동할 수 있습니다. 어려운 문제는 나중에 다시 풀어도 됩니다.

Written 테스트

정의를 보고 영단어를 직접 입력하는 방식입니다.

  1. 영문 정의를 읽습니다.
  2. 해당하는 영단어를 입력창에 타이핑합니다.
  3. Enter 키로 다음 문제로 넘어갑니다.
  4. 모든 문제를 풀면 Submit Test를 클릭합니다.
AI 자동 채점

Written 테스트는 AI가 채점합니다. 약간의 오타는 허용될 수 있지만, 정확하게 입력하는 것이 좋습니다.

통과 기준

선생님이 설정한 점수(보통 95%) 이상을 받아야 통과입니다.

  • 통과: 다음 단계로 진행합니다
  • 미통과: Retake 버튼으로 재시험을 볼 수 있습니다

챌린지 (이의 제기)

Written 테스트에서 AI 채점이 틀렸다고 생각하면 챌린지를 신청할 수 있습니다.

  1. 테스트 결과에서 이의가 있는 문제를 찾습니다.
  2. Challenge 버튼을 클릭합니다.
  3. 챌린지 토큰 1개가 사용됩니다. (총 5개 보유)
  4. 선생님이 검토 후 승인하면 점수가 수정됩니다.
챌린지 토큰

챌린지 토큰은 5개가 주어지고, 사용 후 일정 시간이 지나면 충전됩니다. 신중하게 사용하세요.

6진도 및 통계 확인

단어 상태 이해하기

각 단어는 학습 상태에 따라 다르게 표시됩니다:

상태 의미
NEW 오늘 처음 배운 단어
PASSED 테스트에서 맞춘 단어
FAILED 테스트에서 틀린 단어 (복습 필요)
MASTERED 완전히 익힌 단어 (복습 풀에서 제거)

성적표 (Gradebook)

  1. 상단 메뉴에서 Gradebook을 클릭합니다.
  2. 지금까지의 모든 테스트 기록을 볼 수 있습니다.
  3. 각 테스트를 클릭하면 상세 결과를 확인할 수 있습니다.

블라인드 스팟 (Blind Spots)

블라인드 스팟은 확인이 필요한 단어들입니다:

  • Never Tested: 아직 테스트를 안 본 단어
  • Stale: 21일 이상 확인하지 않은 단어

세션 완료 후 블라인드 스팟이 표시되면 Start Blind Spot Test 버튼을 클릭해서 테스트할 수 있습니다.

마스터된 단어도 확인하세요

마스터된 단어도 21일이 지나면 "Stale" 상태가 됩니다. 블라인드 스팟 체크로 기억을 유지하세요.

7설정 옵션

우측 상단의 톱니바퀴 아이콘을 클릭하면 Settings 페이지로 이동합니다.

테마 설정

옵션 설명
Light Mode 밝은 배경의 기본 테마
Dark Mode 어두운 배경 (눈의 피로 감소)

UI 스타일 설정

항목 옵션
Roundness Sharp (각진) / Normal / Rounded (둥근)
Border Weight Light (얇은) / Normal / Strong (굵은)

학습 진도 초기화

특정 단어장의 학습 기록을 처음부터 다시 시작하고 싶을 때 사용합니다.

  1. Settings 페이지에서 Reset Progress 섹션을 찾습니다.
  2. 클래스와 단어장을 선택합니다.
  3. Reset Progress 버튼을 클릭합니다.
  4. 확인을 위해 RESET을 입력합니다.
  5. 확인 버튼을 클릭하면 초기화됩니다.
주의: 초기화는 되돌릴 수 없습니다

학습 진도를 초기화하면 모든 학습 기록, 테스트 점수, 단어 숙달도가 삭제됩니다. 처음부터 다시 시작해야 합니다.

8자주 묻는 질문

Q. 하루에 꼭 모든 단계를 완료해야 하나요?
아니요. 언제든 나갔다가 다시 이어서 할 수 있습니다. 하지만 테스트 중에 나가면 테스트는 처음부터 다시 봐야 합니다.
Q. 새 단어가 너무 많아서 힘들어요.
vocaBoost가 성적에 따라 자동으로 조절합니다. 점수가 낮으면 새 단어가 줄어들고 복습이 늘어납니다. 선생님께 학습 속도 조절을 요청할 수도 있습니다.
Q. 테스트에서 95% 이상 받기가 너무 어려워요.
통과 기준은 선생님이 설정한 것입니다. 플래시카드로 충분히 학습한 후 테스트를 보면 더 좋은 점수를 받을 수 있습니다.
Q. 마스터된 단어는 더 이상 안 나오나요?
일반 복습에서는 나오지 않지만, 21일이 지나면 "블라인드 스팟"으로 표시됩니다. 블라인드 스팟 체크에서 다시 테스트할 수 있습니다.
Q. 참가 코드를 잘못 입력했어요.
대소문자와 숫자를 정확히 확인하세요. 계속 안 되면 선생님께 코드를 다시 확인해달라고 요청하세요.
Q. 다른 기기에서도 학습할 수 있나요?
네. 같은 계정으로 로그인하면 어디서든 학습할 수 있습니다. 컴퓨터, 태블릿, 스마트폰 모두 가능합니다.
Q. 비밀번호를 잊어버렸어요.
로그인 화면에서 "Forgot Password"를 클릭하고 이메일 주소를 입력하면 비밀번호 재설정 링크가 전송됩니다.
Q. 학습 진도를 초기화하면 어떻게 되나요?
해당 단어장의 모든 학습 기록이 삭제되고 Day 1부터 다시 시작합니다. 이전 테스트 점수도 사라지니 신중하게 결정하세요.
Q. 챌린지 토큰이 부족해요.
챌린지 토큰은 시간이 지나면 자동으로 충전됩니다. 정말 확실할 때만 챌린지를 사용하세요.
Q. 오프라인에서도 사용할 수 있나요?
아니요, vocaBoost는 인터넷 연결이 필요합니다. 학습 진도와 테스트 결과가 실시간으로 저장되기 때문입니다.
