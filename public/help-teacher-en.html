1Getting Started

Creating an Account

  1. Go to the vocaBoost website.
  2. Click the Sign Up button.
  3. Enter your Full Name, email, and password.
  4. Select Teacher for your Role.
  5. Click the Create Account button to complete registration.
Quick Sign Up

Click Continue with Google to quickly sign up with your Google account.

Logging In

  1. Go to the vocaBoost website.
  2. Click the Log In button.
  3. Enter the email and password you used to sign up.
  4. Click the Continue button to go to your dashboard.

2Class Management

Creating a New Class

  1. Find the Classes section on your dashboard.
  2. Click the Create New Class button.
  3. Enter a class name (e.g., "AP History Period 4").
  4. Click Create Class to create the class.
Join Code

When a class is created, a 6-character join code is automatically generated. Students use this code to join your class.

Viewing and Sharing the Join Code

  1. Click on the class card from your dashboard.
  2. Find the join code at the top of the class detail page.
  3. Click the Copy button to copy it to your clipboard.
  4. Share this code with your students.

Viewing Student List

  1. Go to the class detail page.
  2. Click the Students tab.
  3. View all enrolled students' names, emails, and join dates.

Removing Students

  1. In the Students tab, select the checkbox next to the student(s) to remove.
  2. Use Check All to select multiple students.
  3. Click the Remove (N) button.
  4. Click OK in the confirmation dialog to complete.
Note

Removing a student preserves their learning records. If they rejoin, they can continue from where they left off.

Exporting Student List

  1. In the Students tab, select the students you want to export.
  2. Click the Export (N) button.
  3. A CSV file will automatically download.

3Word List Management

Creating a New Word List

  1. Click Lists in the top navigation.
  2. Click the + Create New List button.
  3. Enter a title and description for the word list.
  4. Click Create List.

Adding Words

  1. Click on the word list you want to edit.
  2. Enter the following information:
    • Word: The vocabulary word (e.g., abundant)
    • Part of Speech: Select from the dropdown
    • Definition: English definition
    • Sample Sentence: Example sentence (optional)
    • Secondary Definition: Definition in another language (optional)
  3. Click Add Word.

Editing and Deleting Words

  1. Find the word you want to edit in the word list editor.
  2. To edit: Click the pencil icon, make changes, then click Update Word.
  3. To delete: Click the trash icon and confirm.

Bulk Import from Excel/CSV

  1. In the word list editor, click the Import Words button.
  2. Click Download Template to get the template file.
  3. Fill in your words following the template format.
  4. Select the file or drag and drop it.
  5. Review the preview and click Import N Words.
Template Format

Excel column order: Word | Part of Speech | Definition (EN) | Definition (KO) | Sample

Deleting a Word List

  1. On the Lists page, click the trash icon for the word list you want to delete.
  2. Type the word list name in the confirmation dialog and delete.
Warning

Deleting a word list also deletes all words in it. This action cannot be undone.

4Assigning Word Lists

Assigning a Word List to a Class

  1. Go to the class detail page.
  2. Click the Assigned Lists tab.
  3. Click the Assign List button.
  4. Select the word list to assign.
  5. Configure learning settings and click Assign List.

Learning Settings Options

Setting Description Default
Daily New Words Number of new words students learn per day 20
Test Mode Test type (Multiple Choice Only / Written Only / Both) Multiple Choice
Test Options Number of multiple choice options (4-10) 4
Pass Threshold Minimum score to pass (%) 95%
Study Days/Week Study days per week (1-7) 5
New Word Test Size Number of questions in new word tests 50
Review Test Mode Review test type Multiple Choice
Review Test Size Review test question count (Min ~ Max) 30 ~ 60

Changing Assignment Settings

  1. In the Assigned Lists tab, find the word list you want to modify.
  2. Click the gear (Settings) icon.
  3. Make your desired changes.
  4. Click Save Settings.
Intervention Level

vocaBoost automatically adjusts learning pace based on student performance. If scores are low, new words decrease and review increases.

Unassigning a Word List

  1. In Assigned Lists, find the word list to unassign.
  2. Click the trash icon.
  3. Click OK in the confirmation dialog.
Note

Unassigning a list preserves students' existing learning records.

5Using the Gradebook

Opening the Gradebook

  1. Click Gradebook in the top navigation.
  2. Or click the Open Gradebook button in the Gradebook tab of a class detail page.

Filtering the Gradebook

Use the filters at the top of the gradebook to view specific records:

  • Class: View records for a specific class
  • List: View records for a specific word list
  • Name: View records for a specific student
  • Test Type: View only Multiple Choice or Written tests
  • Date: Today, Yesterday, Past 7 days, Past 30 days, or custom range

Viewing Test Details

  1. Click the View Details button on a test row in the gradebook.
  2. A detail panel opens on the right.
  3. Review each question, the student's answer, and the correct answer.

Exporting Grades

  1. Select the records you want to export using checkboxes.
  2. Click the Export button.
  3. An Excel file (.xlsx) will automatically download.

Reviewing Challenges

When a student disputes an AI-graded answer in a written test, it appears as "Challenge" status.

  1. Find tests with challenge markers in the gradebook.
  2. Click View Details to see the full details.
  3. Review the student's answer and AI reasoning for challenged questions.
  4. Select Accept (approve) or Reject (deny).
Challenge System

Written tests are auto-graded by AI. Students who believe their answer was correct can use a challenge token (5 available) to request a review. If you approve, the score is adjusted and the token is returned.

6FAQ

Q. What if a student loses their join code?
Check the join code on the class detail page and share it with the student again. The join code does not change.
Q. Can I assign the same word list to multiple classes?
Yes, you can assign one word list to multiple classes. You can apply different settings (learning pace, test type, etc.) for each class.
Q. Can I reset a student's learning progress?
Students can reset their own progress from the Settings page. Currently, teachers cannot reset progress on behalf of students.
Q. What's the difference between Multiple Choice and Written tests?
Multiple Choice: Students select the correct answer from options. Fast and intuitive.
Written: Students type the word from a definition. AI-graded, tests more precise recall.
Q. How does Daily New Words work?
The set number of new words is introduced to students each day. However, if a student's scores are low, the system automatically reduces new words and increases review (intervention level adjustment).
Q. What does Pass Threshold mean?
Students must score at or above this percentage to "pass" a test. If set to 95%, they need to get 95 out of 100 correct. Students who don't pass can retake the test.
Q. Does editing words affect student progress?
Editing definitions or sample sentences does not affect student progress. However, deleting a word removes its learning records.
Q. What happens if I delete a class?
All student enrollments for that class are removed. Students' personal learning records are preserved, but they can no longer study through that class.
