1시작하기

계정 만들기

  1. vocaBoost 웹사이트에 접속합니다.
  2. Sign Up 버튼을 클릭합니다.
  3. 이름(Full Name), 이메일, 비밀번호를 입력합니다.
  4. 역할(Role)에서 Teacher를 선택합니다.
  5. Create Account 버튼을 클릭하면 가입이 완료됩니다.
간편 가입

Continue with Google 버튼을 클릭하면 Google 계정으로 빠르게 가입할 수 있습니다.

로그인하기

  1. vocaBoost 웹사이트에 접속합니다.
  2. Log In 버튼을 클릭합니다.
  3. 가입할 때 사용한 이메일과 비밀번호를 입력합니다.
  4. Continue 버튼을 클릭하면 대시보드로 이동합니다.

2클래스 관리

새 클래스 만들기

  1. 대시보드에서 Classes 섹션을 찾습니다.
  2. Create New Class 버튼을 클릭합니다.
  3. 클래스 이름을 입력합니다. (예: "AP History Period 4")
  4. Create Class 버튼을 클릭하면 클래스가 생성됩니다.
참가 코드 (Join Code)

클래스가 생성되면 6자리 참가 코드가 자동으로 만들어집니다. 학생들은 이 코드를 입력해서 클래스에 참가합니다.

참가 코드 확인 및 공유

  1. 대시보드에서 해당 클래스 카드를 클릭합니다.
  2. 클래스 상세 페이지 상단에서 참가 코드를 확인합니다.
  3. Copy 버튼을 클릭하면 클립보드에 복사됩니다.
  4. 이 코드를 학생들에게 공유하세요.

학생 목록 확인

  1. 클래스 상세 페이지로 이동합니다.
  2. Students 탭을 클릭합니다.
  3. 참가한 모든 학생의 이름, 이메일, 참가 일시를 확인할 수 있습니다.

학생 제거하기

  1. Students 탭에서 제거할 학생의 체크박스를 선택합니다.
  2. 여러 학생을 선택하려면 Check All을 사용합니다.
  3. Remove (N) 버튼을 클릭합니다.
  4. 확인 창에서 확인(OK)을 클릭하면 완료됩니다.
참고

학생을 제거해도 해당 학생의 학습 기록은 보존됩니다. 다시 참가하면 기존 진도를 이어갈 수 있습니다.

학생 목록 내보내기

  1. Students 탭에서 내보낼 학생들을 선택합니다.
  2. Export (N) 버튼을 클릭합니다.
  3. CSV 파일이 자동으로 다운로드됩니다.

3단어장 관리

새 단어장 만들기

  1. 상단 네비게이션에서 Lists를 클릭합니다.
  2. + Create New List 버튼을 클릭합니다.
  3. 단어장 제목과 설명을 입력합니다.
  4. Create List 버튼을 클릭합니다.

단어 추가하기

  1. 단어장 목록에서 편집할 단어장을 클릭합니다.
  2. 다음 정보를 입력합니다:
    • Word: 영단어 (예: abundant)
    • Part of Speech: 품사 드롭다운에서 선택
    • Definition: 영문 정의
    • Sample Sentence: 예시 문장 (선택사항)
    • Secondary Definition: 한국어 등 다른 언어 정의 (선택사항)
  3. Add Word 버튼을 클릭합니다.

단어 수정 및 삭제

  1. 단어장 편집 화면에서 수정할 단어를 찾습니다.
  2. 수정: 연필 아이콘을 클릭하고 내용을 변경한 후 Update Word를 클릭합니다.
  3. 삭제: 휴지통 아이콘을 클릭하고 확인합니다.

Excel/CSV로 대량 가져오기

  1. 단어장 편집 화면에서 Import Words 버튼을 클릭합니다.
  2. Download Template을 클릭해서 템플릿 파일을 받습니다.
  3. 템플릿에 맞춰 단어들을 입력합니다.
  4. 파일을 선택하거나 드래그 앤 드롭합니다.
  5. 미리보기를 확인하고 Import N Words 버튼을 클릭합니다.
템플릿 형식

Excel 파일의 열 순서: Word | Part of Speech | Definition (EN) | Definition (KO) | Sample

단어장 삭제하기

  1. Lists 페이지에서 삭제할 단어장의 휴지통 아이콘을 클릭합니다.
  2. 확인 창에서 단어장 이름을 입력하고 삭제합니다.
주의

단어장을 삭제하면 포함된 모든 단어도 함께 삭제됩니다. 이 작업은 되돌릴 수 없습니다.

4단어장 할당

클래스에 단어장 할당하기

  1. 클래스 상세 페이지로 이동합니다.
  2. Assigned Lists 탭을 클릭합니다.
  3. Assign List 버튼을 클릭합니다.
  4. 할당할 단어장을 선택합니다.
  5. 학습 설정을 지정하고 Assign List를 클릭합니다.

학습 설정 옵션

설정 설명 기본값
Daily New Words 학생이 하루에 배울 새 단어 수 20개
Test Mode 테스트 유형 (Multiple Choice Only / Written Only / Both) Multiple Choice
Test Options 객관식 선택지 개수 (4~10) 4개
Pass Threshold 통과 기준 점수 (%) 95%
Study Days/Week 주당 학습 일수 (1~7) 5일
New Word Test Size 신규 단어 테스트 문제 수 50개
Review Test Mode 복습 테스트 유형 Multiple Choice
Review Test Size 복습 테스트 문제 수 (Min ~ Max) 30 ~ 60개

할당 설정 변경하기

  1. Assigned Lists 탭에서 설정을 변경할 단어장을 찾습니다.
  2. 톱니바퀴(Settings) 아이콘을 클릭합니다.
  3. 원하는 설정을 변경합니다.
  4. Save Settings를 클릭합니다.
개입 수준 (Intervention Level)

vocaBoost는 학생의 성적에 따라 자동으로 학습량을 조절합니다. 점수가 낮으면 새 단어 수가 줄어들고 복습 비중이 늘어납니다.

단어장 할당 해제

  1. Assigned Lists에서 해제할 단어장을 찾습니다.
  2. 휴지통 아이콘을 클릭합니다.
  3. 확인 창에서 확인(OK)을 클릭합니다.
참고

할당을 해제해도 학생들의 기존 학습 기록은 보존됩니다.

5성적표 활용

성적표 열기

  1. 상단 네비게이션에서 Gradebook을 클릭합니다.
  2. 또는 클래스 상세 페이지에서 Gradebook 탭의 Open Gradebook 버튼을 클릭합니다.

성적표 필터링

성적표 상단의 필터를 사용하여 원하는 기록만 볼 수 있습니다:

  • Class: 특정 클래스의 기록만 보기
  • List: 특정 단어장의 기록만 보기
  • Name: 특정 학생의 기록만 보기
  • Test Type: Multiple Choice 또는 Written 테스트만 보기
  • Date: Today, Yesterday, Past 7 days, Past 30 days, 또는 직접 범위 지정

테스트 상세 보기

  1. 성적표에서 확인할 테스트 행의 View Details 버튼을 클릭합니다.
  2. 오른쪽에 상세 패널이 열립니다.
  3. 각 문제별로 학생의 답변과 정답을 확인할 수 있습니다.

성적 내보내기

  1. 내보낼 기록들을 체크박스로 선택합니다.
  2. Export 버튼을 클릭합니다.
  3. Excel 파일(.xlsx)이 자동으로 다운로드됩니다.

챌린지 검토하기

학생이 타이핑 테스트에서 채점 결과에 이의를 제기하면 "Challenge" 상태로 표시됩니다.

  1. 성적표에서 챌린지 표시가 있는 테스트를 찾습니다.
  2. View Details를 클릭해서 상세 내용을 확인합니다.
  3. 챌린지된 문제에서 학생의 답변과 AI 추론을 검토합니다.
  4. Accept(승인) 또는 Reject(거절)을 선택합니다.
챌린지 시스템

타이핑 테스트는 AI가 자동 채점합니다. 학생이 자신의 답변이 맞다고 생각하면 챌린지 토큰(5개 보유)을 사용해 재검토를 요청할 수 있습니다. 교사가 승인하면 점수가 수정되고 토큰이 반환됩니다.

6자주 묻는 질문

Q. 학생이 참가 코드를 분실했다면?
클래스 상세 페이지에서 참가 코드를 다시 확인하고 학생에게 알려주세요. 참가 코드는 변경되지 않습니다.
Q. 단어장을 여러 클래스에 할당할 수 있나요?
네, 하나의 단어장을 여러 클래스에 각각 할당할 수 있습니다. 클래스마다 다른 설정(학습 속도, 테스트 유형 등)을 적용할 수 있습니다.
Q. 학생의 학습 진도를 초기화할 수 있나요?
학생 본인이 Settings 페이지에서 직접 초기화할 수 있습니다. 교사가 대신 초기화하는 기능은 현재 제공되지 않습니다.
Q. Multiple Choice와 Written 테스트의 차이는?
Multiple Choice: 선택지 중 정답을 고릅니다. 빠르고 직관적입니다.
Written: 정의를 보고 단어를 직접 입력합니다. AI가 채점하며, 더 정확한 암기력을 테스트합니다.
Q. Daily New Words는 어떻게 작동하나요?
설정한 숫자만큼의 새 단어가 매일 학생에게 소개됩니다. 단, 학생의 성적이 낮으면 시스템이 자동으로 새 단어 수를 줄이고 복습을 늘립니다(개입 수준 조절).
Q. Pass Threshold의 의미는?
학생이 테스트에서 이 점수 이상을 받아야 "통과"로 처리됩니다. 95%로 설정하면 100개 중 95개 이상 맞춰야 합니다. 통과하지 못하면 재시험을 볼 수 있습니다.
Q. 단어를 수정하면 학생 진도에 영향이 있나요?
단어의 뜻이나 예시 문장을 수정해도 학생의 학습 진도에는 영향이 없습니다. 단어 자체를 삭제하면 해당 단어의 학습 기록도 삭제됩니다.
Q. 클래스를 삭제하면 어떻게 되나요?
클래스와 관련된 모든 학생 연결이 해제됩니다. 학생들의 개인 학습 기록은 유지되지만, 해당 클래스를 통한 학습은 더 이상 진행할 수 없습니다.
